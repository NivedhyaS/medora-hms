# Medora Hospital Management System (HMS)

Medora HMS is a web-based application developed to streamline and digitalize hospital operations, including patient management, appointments, billing, and laboratory reports.

---

## Live Demo
https://medora-hms-main-t4wedq.free.laravel.cloud/

## GitHub Repository
https://github.com/NivedhyaS/medora-hms.git

---

## Features
- Patient registration and record management  
- Appointment scheduling with doctor availability  
- Role-based access control (Admin, Doctor, Staff)  
- Billing and payment processing  
- Laboratory report management  
- Secure authentication system  

---

## Technology Stack
- Backend: Laravel (PHP)  
- Frontend: HTML, CSS, JavaScript, Bootstrap  
- Database: PostgreSQL  
- Architecture: MVC

## About the Project
This project was developed to improve efficiency in hospital management by integrating all core functionalities into a single system. It demonstrates practical implementation of MVC architecture, CRUD operations, authentication, and database design.
